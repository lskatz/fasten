pub struct Writer<W: Write> { /* fields omitted */ }
Expand description

Writes bgzip file.

You can create a writer using from_path or using WriterBuilder. Additional threads are used to compress blocks, while the main thread reads the writes to a file/stream. If additional_threads is 0, the main thread will compress blocks itself.

It is highly not recommended to continue writing after an error, as in that case the writer may miss some blocks.
